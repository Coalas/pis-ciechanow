Falling Snow Example B for jSnow v1.2.

In this example we create (via jScript) a button that gives the user the option to turn on and off the jSnow plugin. For the implementation I use the toggle event of jQuery, I hope you understand it.

Code used in this example of jSnow:

<script type="text/javascript">
var jSnow;
var jSnowToggle = $("<button/>");
  jSnowToggle.html("START snowing");
jSnowToggle.toggle(snow,stopSnow);
function snow()
{
   $(this).text("STOP snowing");
   jSnow = $().jSnow();
}
function stopSnow()
{
   $(this).text("START snowing");
   jSnow.kill();
}
$(function() {
   $("#jSnowToggleContainer").append(jSnowToggle);
});
</script>
